# Deio Scroll brand system

Deio is the product family. **Scroll** is the motion-capture product: it turns a
live website into a deliberately paced, continuous video.

## Identity

The Deio Scroll mark combines a structural **D** with a curved scroll path. The
blue point is the active playhead; the cyan line is the captured trajectory.
Use the complete `Deio Scroll` name in titles and documentation. In compact
navigation, set `Deio` as the wordmark and `SCROLL` as the product label.

Primary logo assets:

- `assets/deio-scroll-mark.svg` — repository and documentation
- `apps/web/public/deio-scroll-mark.svg` — product UI and favicon

Keep clear space around the mark equal to one quarter of its width. Do not
recolor individual parts or place it directly over detailed imagery.

## Palette

| Token | Dark | Light | Role |
| --- | --- | --- | --- |
| Canvas | `#0D0F0C` | `#EEECE3` | Application background |
| Surface | `#151713` | `#FAF8F1` | Cards, navigation, controls |
| Raised surface | `#1E211C` | `#E6E3D8` | Inputs and secondary layers |
| Primary text | `#F4F1E8` | `#171914` | Main copy and structure |
| Muted text | `#9B9D92` | `#5F6259` | Labels and metadata |
| Cobalt blue | `#6B8CFF` | `#3158C9` | Primary actions and playhead |
| Motion cyan | `#5AD8C5` | `#087E72` | Trajectory and directional cues |

Blue communicates action: analyze, capture, selection, and active progress.
Cyan communicates motion: scroll paths, spatial cues, and ambient depth. Avoid
using both as competing calls to action.

## Typography

- **Syne** — wordmark and high-impact display moments
- **Instrument Sans** — interface, controls, labels, and documentation UI

Use compact display type sparingly. Deio Scroll is a precision tool, so the
interface should remain calm, dense, and highly legible.
